# Study Buddy

## Backend

1. **Setup:**

   ```bash
   cd backend
   npm i

   ```

2. **Run Backend:**
   ```bash
   npm run dev
   ```

## Frontend

### On Another Terminal

1. **Setup:**

   ```bash
   cd frontend
   npm i

   ```

2. **Run Frontend:**
   ```bash
    npm start
   ```

```bash
    Frontend: http://localhost:5173
    Backend: http://localhost:4000
```
